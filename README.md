### 🔹 ResumeMatch AI – Intelligent Resume Screening System  
🔗 https://github.com/Vishnu-Shettihalli/ResumeMatch-AI  

**📌 Description:**  
An NLP-powered system that evaluates and ranks resumes based on job descriptions. It uses TF-IDF vectorization and cosine similarity to identify the best candidate matches efficiently.

**⚙️ Tech Stack:**  
- Python  
- Scikit-learn (TF-IDF)  
- NLTK  
- PDF Processing  
- Streamlit
